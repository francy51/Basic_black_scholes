# Options Pricing - Python & Rust Implementation

A complete options pricing toolkit with data downloading (Python) and Black-Scholes analysis (Rust).

## Project Structure

```
options_pricing/
├── download_options.py      # Python script for downloading option chain data
├── src/main.rs              # Rust program for Black-Scholes pricing
├── requirements.txt         # Python dependencies
├── Cargo.toml               # Rust dependencies
└── README.md                # This file
```

## Python Script (Data Download)

### Features
- Downloads complete option chains from Yahoo Finance
- Calculates all Black-Scholes model inputs:
  - Current stock price (S)
  - Risk-free rate (r) - from Treasury yields
  - Historical volatility (σ) - from 1-year daily returns
  - Dividend yield (q)
  - Time to expiration (T)
- Compares implied volatility to historical volatility
- Exports data to CSV with all BS parameters included

### Usage

```bash
# Setup
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# Run
python3 download_options.py AAPL
python3 download_options.py TSLA
python3 download_options.py SPY
```

### Output Files
- `{TICKER}_{DATE}_{TIMESTAMP}_calls.csv` - Call options with BS parameters
- `{TICKER}_{DATE}_{TIMESTAMP}_puts.csv` - Put options with BS parameters
- `{TICKER}_{DATE}_{TIMESTAMP}_black_scholes_params.csv` - BS parameters only

## Rust Program (Black-Scholes Analysis)

### Features
- Custom Black-Scholes implementation with dividend support
- Reads CSV files generated by Python script
- Calculates theoretical option prices
- Compares market vs theoretical prices
- Identifies potential mispricing opportunities
- Fast performance with release builds

### Usage

```bash
# Build
cargo build --release

# Run
./target/release/options_pricing AAPL_2026-03-02_20260228_113854_calls.csv
./target/release/options_pricing TSLA_2026-03-02_20260228_000641_calls.csv
```

### Output
- Black-Scholes parameters summary
- Option chain with market vs theoretical prices
- At-the-money option analysis
- Pricing difference percentages

## Workflow

1. **Download Data (Python)**
   ```bash
   python3 download_options.py AAPL
   ```

2. **Analyze with Rust**
   ```bash
   ./target/release/options_pricing AAPL_2026-03-02_*.csv
   ```

3. **Review Results**
   - Market price vs theoretical price
   - Identify mispricing opportunities
   - Compare IV to historical volatility

## Black-Scholes Formula

### Call Option
```
C = S·e^(-qT)·N(d1) - K·e^(-rT)·N(d2)
```

### Put Option
```
P = K·e^(-rT)·N(-d2) - S·e^(-qT)·N(-d1)
```

### Where
```
d1 = [ln(S/K) + (r - q + σ²/2)T] / (σ·√T)
d2 = d1 - σ·√T
```
